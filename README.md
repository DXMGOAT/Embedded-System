基于 STM32 + FPGA 的数字基带调制解调与误码率测试系统，分为STM32和FPGA两块。

实验概况见 实验要求.jpg/txt

系统运行流程：STM32 发送 → FPGA 调制 → 信道回传 → STM32 解调 → LCD 显示

系统运行请观看 模拟测试视频.mp4及 加入FPGA传输测试视频.mp4

系统实现内容：

STM32部分：首先完成基于 STM32 的数据基带传输系统设计。通过 STM32 模拟数据源，实现数据生成、处理以及基带信号传输功能。
              
具体实现：STM32外设初始化与系统搭建；模拟数据生成模块设计；基带数据处理流程实现；数据发送接口设计与调试。通过串口等通信接口验证数据传输过程，确保基带数据能够稳定输出。

FPGA部分：基于 FPGA 平台，采用 Verilog HDL 完成数字逻辑系统设计，实现数据传输、缓存以及调制功能。

具体实现：FPGA通信接口设计；SPI数据接收模块设计；FIFO数据缓存模块设计；数据调制处理模块设计；FPGA综合、仿真及硬件验证。

联合调试测试部分：
STM32数据生成与发送功能；
FPGA数据接收与处理功能；
数据传输稳定性；
系统整体运行效果。

实验存在的问题：1、眼图效果不理想 2、连通FPGA后的调制解调波形与模拟出来的差距较大。
